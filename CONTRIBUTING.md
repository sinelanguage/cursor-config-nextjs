# Contributing

Thanks for contributing to the Next.js Cursor Config!

## Development Setup

```bash
npm install
npm run lint
npm run type-check
npm run test
npm run build
```

## Standards

- Strict TypeScript (no `any`)
- App Router-first for new patterns
- Route Handlers for API/BFF
- Use Server Components by default
- Accessibility and security are mandatory

## Commit Convention

Use Conventional Commits:

```text
feat(scope): summary
fix(scope): summary
docs(scope): summary
```
